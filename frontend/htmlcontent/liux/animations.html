Animations

Basic Usage

To Use the animations on your layout add class .animated-nameOfAnimation or .animated-nameOfAnimation--Variation (if it has a variation) to your HTML element depending on what kind of animation works for your element. You can trigger the animations in two ways:

  • 1. Put Class directly on html 
    					
					<i class="icon-star animated-bounce--infinite" title="icon"></i>
  • 2. Call it Dynamically using JQuery 
    					
						$("button").click(function () {
						$(this).addClass("animated-rotate--left")
						});

Types and Classes

Call to action Animations

Use it for Call to Action elements like <button>, <a> or <icon>, also with important HTML elements to communicate a call to "click me". You can use it as a One- time or Infinite animation.

Please do not use on container HTML elements like <div> or <section>.

  • Bounce

    Smooth Jumping Element

    .animated-bounce

    .animated-bounce--infinite

  • Flash

    Bright light to illuminate elements

    .animated-flash

    .animated-flash--infinite

  • Pulse

    Beating simulation efect

    .animated-pulse

    .animated-pulse--infinite

  • Shake

    Smooth Left and right movement

    .animated-shake

    .animated-shake--infinite

Interaction

Use it for interactions with HTML Container elements like <div> or <section> and some inline elements like <image> or <p>.

Should be use after an event or a click, to determinates another element's state for the action that user is making.

  • Flip left

    3D Rotation for the element from right to left

    .animated-flip--left

  • Flip up

    3D Rotation for the element from down to up

    .animated-flip--up

  • Fade in

    Smooth change of opacity's element from transparent to solid fill

    .animated-fade--in

  • Fade out

    Smooth change of opacity's element from solid fill to transparent

    .animated-fade--out

  • Zoom in

    Scale the element size to big

    .animated-zoom--in

  • Zoom out

    Scale the element size to small

    .animated-zoom--out

Whirl Animations

Can be used for Interactions or Elements that need a loop.

  • Rotate left

    2D rotation to left

    .animated-rotate--left

  • Rotate right

    2D rotation to right

    .animated-rotate--right

  • Spin

    Infinite loop in 2D rotation for an element

    .animated-spin--infinite

Variations

As shown on the class with variation .animated-nameOfAnimation--Variation this variations can be defined by Timing and Direction.

  • 1. Timing - infinite 
    					
					<i class="icon-star animated-spin--infinite" title="icon"></i>
  • 2. Direction - left, right, up, in, out. 
    					
						<i class="icon-star animated-rotate--left" title="icon"></i>
