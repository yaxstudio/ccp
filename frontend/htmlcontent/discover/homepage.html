Library Initiatives

Below are the current projects Laureate libraries are working collaboratively to enhance student experience.

EBSCO Discovery Service

Fast and dynamic single search box for all academic content including books, e-books, journal articles and references. We have negotiated discounted prices for network institutions to implement in their Library Portals. Please contact us for further information

Network Library Index Project

Assuring that our libraries are offering the best possible service to our students and faculty has always been a priority for us. The Network Library Index initiative is a project proposed by Universidad Europea de Madrid to collect and maintain a set of library quality indicators all Laureate institutions can agree to participate. These indicators are developed based on the European and American library standards. To find out more about this project and how to participate please contact the Discover team.

OCLC QuestionPoint

Answer questions anywhere, anytime through cooperative virtual reference. QuestionPoint provides a complete virtual reference management system, integrating chat, e-mail, a reference knowledge base, reports and analytic tools to give you a complete view of reference activity. Expand your library hours of service to offer live help anytime through membership in the 24/7 Reference® Cooperative. We have negotiated discounted prices for network institutions to implement in their Library Portals. Please contact us for further information

Publishers Contracts

If you would like to learn more about what current contracts are under negotiation or if you have a content provider you believe we should invite to learn more about their offerings, please contacts us directly to access our community of practice portal and vendor’s status.
